# 🦵 腿部露出检测工具 - GUI 使用说明

## 快速启动

```bash
python leg_detector_gui.py
```

## 界面功能介绍

### 1. 视频选择区
- 点击"浏览视频"按钮选择要处理的视频文件
- 支持格式：MP4, AVI, MOV, MKV, FLV, TS, WMV

### 2. 参数配置区

| 参数 | 说明 | 推荐值 |
|------|------|--------|
| 模型 | 选择检测精度模型 | models/yolov8m-pose.pt (推荐) |
| 最小时长 | 提取的片段最短长度(秒) | 2-5秒 |
| 采样间隔 | 每隔多少秒采样一次 | 2秒(快速)/1秒(精确) |
| 置信度阈值 | 关键点检测的最小置信度 | 0.5-0.6 |
| 硬件加速 | 选择处理设备 | cuda (GPU加速) / cpu |
| 输出目录 | 结果保存路径 | output/leg_output |

### 3. 操作按钮

| 按钮 | 功能 |
|------|------|
| 🚀 开始检测 | 启动视频检测处理 |
| 📂 打开输出目录 | 查看生成的结果文件 |

### 4. 处理进度区
- 进度条显示处理状态
- 实时显示当前处理进度百分比

### 5. 日志输出区
- 显示处理过程中的详细日志信息
- 包含时间戳，方便追踪

## 使用流程

### 1. 选择视频
1. 点击"浏览视频"按钮
2. 在弹出的文件选择框中选择你的视频文件

### 2. 配置参数
- **模型选择**：根据需要选择精度（n/s/m/l/x）
  - yolov8n-pose.pt：最快，但精度较低
  - yolov8m-pose.pt：平衡推荐
  - yolov8l-pose.pt：高精度

- **最小时长**：2-5秒之间比较合适
- **采样间隔**：3秒平衡推荐
- **置信度阈值**：0.5-0.6推荐
- **硬件加速**：cuda（推荐）/ mps / cpu

### 3. 开始检测
1. 确认视频和参数无误
2. 点击"🚀 开始检测"按钮
3. 等待处理完成

### 4. 查看结果
- 结果会自动保存到输出目录
- 点击"📂 打开输出目录"直接打开结果文件夹

## 输出文件说明

```
output/leg_output/
└── video_name/
    ├── leg_clip_*.mp4
    ├── leg_processing_info.json
    └── ...
```

## 常见问题

### Q: 处理速度太慢怎么办？
A: 增大采样间隔、使用较小模型、使用GPU加速

### Q: 漏检了一些片段？
A: 降低置信度阈值、减小采样间隔、使用更大模型

### Q: GPU加速不可用？
A: 需要安装CUDA版PyTorch：
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu121
```
